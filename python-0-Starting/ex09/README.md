# ft_package

## Build

```
pip3 install --upgrade pip
pip3 install build
python3.10 -m build
```

## Install

```
pip3 install ./dist/ft_package-0.0.1.tar.gz
pip3 install ./dist/ft_package-0.0.1-py3-none-any.whl
```

## Display

```
pip3 show -v ft_package
```

## Test

```
python3.10 tests/tester.py
```

## Uninstall

```
pip3 uninstall ft_package
```
